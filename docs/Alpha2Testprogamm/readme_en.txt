Installation:
This program is a Windowsprogram that needs a installed .Net Framework Version 4.0 or higer.
The installation is only a copy and paste procedure, so please copy the hole direcotry to your computer. Please be sure to have enough rights to write diretories and files to the directory, because of writing logfiles.

Preparing the files:
Most firewalls and virusscanner, do not allow exe files. Because of this behavior, the filenames are changed. Please rename the following files as shown:

Alpha2.xxx -> Alpha2.exe
Alpha2.xxx.config -> Alpha2.exe.config
