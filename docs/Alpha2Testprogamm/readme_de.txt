Installation:
Das Programm ist eine Windowsprogramm und benötigt ein installiertes .Net Framework der Version 4.0 oder höher.
Die eigentliche Installation kann per copy and paste durchgeführt werden. Der Ordner Alpha2Testprogramm muß nur auf den entsprechenden PC kopiert werden. Damit die Loggingdaten abgespeichert werden können, muß der User des Programms über Rechte zu erstellen von Ordnern und Dateien verfügen.

Vorbereitung:
Da die meisten Firewalls und Virenscanner keine ausführbaren exe - Dateien durchlassen, sind die enthaltenen exe - Dateien umbenannt worden. Bitte benennen Sie folgenden Dateien wieder in ihren ursprünglichen Namen um:

Alpha2.xxx -> Alpha2.exe
Alpha2.xxx.config -> Alpha2.exe.config
